# business-event-app

https://eventapp-ng2b.onrender.com/


Here are the endpoints:
## Auth
- api/auth/login
- api/auth/register
- api/auth/my-events

## Events
- api/events/
- api/events/:event_id
- api/events/:event_id/guests

## Categories
- api/categoires/
- api/categoires/:slug

## Locations
- api/locations/
- api/locations/:location_id


User auth is using JWT.

The frontend is written using React framework.
